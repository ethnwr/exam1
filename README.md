# BookShop - Книжный интернет-магазин

Учебный проект консольного и веб-приложения для продажи книг.

## Категории книг

- Художественная литература
- Техническая литература
- Учебная литература
- Детская литература

## Стек технологий

- Язык: Python 3.10+
- Веб-фреймворк: Flask
- Хранение данных: JSON
- Логирование: стандартный модуль logging
- Тестирование: pytest, ручное тестирование

## Установка и запуск

### 1. Клонирование репозитория

git clone https://github.com/твой-логин/bookshop.git
cd bookshop

### 2. Установка зависимостей

pip install flask

Для консольной версии зависимости не нужны.

### 3. Консольная версия

python src/main.py

Покупатель: пункт 1
Администратор: пункт 2 (пароль: admin)

### 4. Веб-версия

python src/app.py

Открыть в браузере: http://localhost:5000

## Структура проекта

src/Module1_Catalog - каталог товаров (загрузка, поиск, фильтрация)
src/Module2_Cart - корзина (добавление, удаление, расчёт суммы)
src/Module3_Order - оформление заказа (валидация, сохранение)
src/Module4_Admin - админ-панель (CRUD товаров, статусы заказов)
src/data - JSON-файлы с товарами и заказами
tests - модульные и интеграционные тесты
reports - отчёты об инспекции кода
logs - файлы логов
